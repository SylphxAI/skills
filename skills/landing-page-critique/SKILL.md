# Landing Page Critique

Use this skill to turn a landing page into a clear conversion argument.

## Workflow

1. Identify audience, traffic source, product maturity, and desired conversion.
2. Read `references/landing-page-patterns.md`.
3. Evaluate message, proof, product demonstration, CTA path, objections, trust, and measurement.
4. Separate positioning problems from UI polish problems.
5. Produce prioritized copy/layout experiments.

## When not to use

- Do not use for generic advice the base model already handles without this skill's specific artifact contract.

## Guardrails

- Do not optimize persuasion by misleading users.
- Do not add testimonials, logos, or claims that cannot be substantiated.
- Treat SEO as distribution support, not a reason to dilute the core promise.
