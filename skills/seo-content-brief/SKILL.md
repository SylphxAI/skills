# SEO Content Brief

Use this skill to create content briefs that satisfy search intent and advance the product story.

## Workflow

1. Identify target audience, search intent, business goal, and product relevance.
2. Read `references/seo-content-brief-patterns.md`.
3. Map query cluster, SERP expectations, differentiators, outline, proof, internal links, and conversion path.
4. Define what original value the page adds beyond existing results.
5. Produce a brief with measurement and refresh plan.

## When not to use

- Do not use for generic advice the base model already handles without this skill's specific artifact contract.

## Guardrails

- Do not create thin content that repeats search results.
- Do not force product CTAs where the intent is informational unless there is a natural next step.
- Do not invent stats, quotes, or comparisons.
